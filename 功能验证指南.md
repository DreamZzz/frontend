## 故障排除指南

### 问题1: 中文输入仍无法正常工作
**可能原因**: TextInput属性仍需调整
**解决方案**: 
1. 检查 `CreatePostScreen.js` 和 `DetailScreen.js` 中的TextInput组件
2. 尝试添加 `editable={true}` 属性
3. 尝试移除 `textAlignVertical="top"` 属性

### 问题2: 评论仍显示500错误
**调试步骤**:
1. 查看前端控制台输出:
   ```bash
   # 在Metro Bundler终端查看日志
   cd /Users/qiang/social-app-ios/frontend
   npm start
   ```
2. 查看后端日志:
   ```bash
   tail -f /Users/qiang/social-app-ios/backend/logs/app.log
   ```
3. 检查API调用格式:
   - 打开浏览器开发者工具 → Network标签
   - 查看评论请求的请求体和参数

### 问题3: 返回按钮仍显示英文
**检查项目**:
1. 确认 `AppNavigator.js` 中三个屏幕都设置了 `headerBackTitle: '返回'`
2. 检查iOS系统语言设置为中文
3. 尝试清理应用缓存并重新启动
